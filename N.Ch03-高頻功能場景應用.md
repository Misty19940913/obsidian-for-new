# N.Ch03｜高頻功能場景應用

> [!info]+ 這章結束時你會做到的事
> 1. 掌握快速切換、搜尋、命令面板的核心用法
> 2. 知道什麼情況下用哪個功能
> 3. 建立使用 Graph View 的習慣

---

## 場景一：想不起來筆記叫什麼名字

**你遇到的事：** 「我知道有篇關於習慣的筆記，但不知道存在哪裡。」

**Obsidian 怎麼做：** 快速切換器（Quick Switcher）

按 `Ctrl/Cmd + O`，輸入「習慣」——不管筆記叫「原子習慣」還是「習慣養成」，只要名字包含「習慣」就會出現。

**進階技巧：**

| 輸入 | 效果 |
|------|------|
| `#目標` | 搜尋含 `#目標` 標籤的筆記 |
| `>`開頭 | 搜尋命令 |
| `/` 開頭 | 跳轉到資料夾 |
| `2026*` | 搜尋所有以 2026 開頭的筆記 |

---

## 場景二：想找某句話在哪篇筆記裡

**你遇到的事：** 「我在某篇筆記寫過『第二曲線』，但忘了是哪篇。」

**Obsidian 怎麼做：** 全域搜尋

按 `Ctrl/Cmd + Shift + F`，輸入「第二曲線」——所有筆記、所有位置全部搜一遍，1 秒找到。

> [!tip]+ 搜尋語法
> `word1 AND word2` → 同時包含兩個關鍵字
> `tag:#讀書` → 只搜含 `#讀書` 標籤的筆記
> `path:folder` → 在特定資料夾搜尋

---

## 場景三：不知道某個功能在哪裡

**你遇到的事：** 「想要把這篇筆記匯出成 PDF，但找不到匯出選項。」

**Obsidian 怎麼做：** 命令面板

按 `Ctrl/Cmd + P`（或 `Ctrl/Cmd + Shift + P`），輸入「匯出」——所有相關命令列出來，點一下就執行。

不需要記快速鍵，**記這個捷徑就好**：`Ctrl/Cmd + P`。

---

## 場景四：想看這篇筆記和哪些筆記有關聯

**你遇到的事：** 「這篇筆記寫完了，想確認有沒有遺漏應該連結的內容。」

**Obsidian 怎麼做：** 本地圖譜（Local Graph）

按 `Ctrl/Cmd + G`，在筆記內直接顯示這篇周圍的連結網絡。拖曳、雙擊節點跳轉。

> [!tip]+ 什麼時候看圖譜
> 不是每天都看。當你寫完一篇較長的筆記，想重新檢視哪些想法應該連結時，打開看一次。

---

## 場景五：某個搜尋未來會重複執行

**你遇到的事：** 「每週都要找一次『這週新建的所有筆記』，懶得每次都重新搜。」

**Obsidian 怎麼做：** 書籤（Bookmarks）

1. 執行一次搜尋（如 `tag:#目標`）
2. 點「儲存」→ 輸入書籤名稱
3. 側邊欄書籤區，以後點一下就執行同樣搜尋

---

## 常用快捷鍵速查

| 動作 | Windows | macOS |
|------|---------|-------|
| 新建筆記 | `Ctrl + N` | `Cmd + N` |
| 快速切換 | `Ctrl + O` | `Cmd + O` |
| 保存 | `Ctrl + S` | `Cmd + S` |
| 全域搜尋 | `Ctrl + Shift + F` | `Cmd + Shift + F` |
| 命令面板 | `Ctrl + P` | `Cmd + P` |
| 本地圖譜 | `Ctrl + G` | `Cmd + G` |
| 插入連結 | `Ctrl + K` | `Cmd + K` |
| 切換預覽 | `Ctrl + E` | `Cmd + E` |
