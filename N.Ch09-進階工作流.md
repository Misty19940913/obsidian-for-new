## 一、Canvas 畫布：視覺化思考的殺手級工具

Canvas 讓你可以用「無限白板」的方式呈現筆記、圖片、PDF 之間的關聯。它打破了純文字的線性限制。

### 如何開啟 Canvas？
在命令面板（`Ctrl/Cmd + P`）輸入 `Canvas: Create new canvas`，或點擊左側邊欄的 Canvas 圖示。

### 核心應用場景：專案策劃

你可以把零散的筆記拖曳到白板上，畫線連接，理清邏輯。

```text
┌─────────────────────────────────────────┐
│  [Kickoff 會議]  →  [執行監控]  →  [驗收]   │
│       ↓                 ↓              │
│    [任務 1]          [任務 2]          │
└─────────────────────────────────────────┘
```

**小技巧：** 
- 你可以直接在 Canvas 上新增筆記，也可以把現有的 `.md` 檔案從左側拖曳進來。
- 畫布上的卡片會與原始筆記保持同步更新！

## 二、網頁剪藏（Web Clipper）

看到好的網路文章，如何快速存入 Obsidian？

### 推薦工具
- **Obsidian Web Clipper**：官方推出的瀏覽器擴充套件（Chrome / Edge / Safari/ Firefox 均支援）。

### 剪藏格式建議
不要只是把全文塞進去，這會變成「資訊垃圾場」。建議在文章頂部加入你的**主動思考**：

```markdown
---
source: {{url}}
tags:
- 剪藏
- 待讀
---

# 標題

> **為何收藏？這篇文章對我有什麼用？**
> （在這裡寫下一兩句你的想法）

## 摘要與後續行動
- [ ] 閱讀全文
- [ ] 摘錄重點

---
(以下為剪藏的網頁全文)
```

## 三、PDF 與文獻閱讀工作流

如果你需要閱讀大量 PDF 文獻，Obsidian 也能勝任。

### 基本 PDF 閱讀
1. 將 PDF 檔案拖入 Vault 中。
2. 在筆記中用 `![[你的文件.pdf]]` 即可嵌入並直接閱讀。

### 推薦工作流：Canvas 分離閱讀法
將 PDF 拖入 Canvas 左側，右側放幾張空白筆記卡片。邊看 PDF 邊在右側摘錄重點，做到「左邊輸入、右邊輸出」。

### 提問式閱讀：HQ&A 筆記法
每讀完一個段落，強迫自己問三個問題並記錄：
- **H（Highlight）**：這段的重點是什麼？
- **Q（Question）**：這段讓我想到什麼新問題？
- **A（Answer）**：我如何將這個知識應用在我的專案中？
